# Feynman Diagrams

## Graphical Representation of Interactions

Feynman diagrams provide a powerful graphical method for calculating scattering amplitudes in quantum field theory. Each diagram corresponds to a mathematical expression involving propagators and vertices.

### Basic Elements

The fundamental building blocks of Feynman diagrams are:

1. **External Lines** - Represent incoming and outgoing particles
2. **Internal Lines** - Represent virtual particle exchanges (propagators)
3. **Vertices** - Represent interaction points

### Scalar Field Example

For a scalar field with interaction Lagrangian:

$$\mathcal{L}_{\text{int}} = -\frac{\lambda}{4!} \phi^4$$

the vertex factor contributes $-i\lambda$ to each interaction point.

## Propagators and Green's Functions

The scalar propagator represents the amplitude for a virtual particle to travel from one spacetime point to another. In momentum space, it takes the form:

$$D_F(k) = \frac{i}{k^2 - m^2 + i\epsilon}$$

where the $i\epsilon$ prescription ensures proper causal behavior.

### Physical Interpretation

The propagator can be understood as the quantum mechanical amplitude for a particle created at point $x$ to be detected at point $y$. The denominator $k^2 - m^2$ vanishes when the virtual particle goes "on-shell," leading to the characteristic resonance behavior.

## Calculation Rules

To calculate a scattering amplitude from a Feynman diagram:

1. **Assign momentum** to each line, conserving momentum at vertices
2. **Write propagator factors** $i/(k^2 - m^2 + i\epsilon)$ for internal lines  
3. **Include vertex factors** $-i\lambda$ for each interaction point
4. **Integrate** over undetermined internal momenta:
   $$\int \frac{d^4k}{(2\pi)^4}$$
5. **Apply external state normalization** factors

### Simple Example: $2 \to 2$ Scattering

Consider the tree-level scattering process $\phi + \phi \to \phi + \phi$ through a $\phi^4$ interaction:

```
    p₁ \     / p₃
        \   /
         \ /
          X  ← vertex
         / \
        /   \
    p₂ /     \ p₄
```

The amplitude is simply:
$$\mathcal{M} = -i\lambda$$

This demonstrates how Feynman diagrams encode both the physics and the mathematical structure of quantum field interactions.
